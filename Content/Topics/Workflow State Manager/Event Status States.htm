For Slot Sets the following Status States may be configured for different permissions.

  • Pending – The Event Slot Set has not reached the Status.
    Note: User Rights don’t need to be configured for this Status State.
  • In Progress – The Event Slot Set is being worked upon.

  • Completed – The Event Slot Set has completed the expected work for the timeframe of the Status.
