Price Action Status States

Price Action Status States are determined by configured Validations that system evaluates against the Price Action during the time of its calculation. For more information on configuring Price Action Validations and viewing failed Validation information, see Create Pricing Plans and Actions.

  • Validated – All Validations run against the Price Action were successful.
  • Warning(s) – At least one Price Action Validation failed that was configured with the ‘Warning’ severity level.
  • Error(s) – At least one Price Action Validation failed that was configured with the ‘Error’ severity level.

Note: To ease configuration, Price Actions Validation Status State type ‘Information’ is included with ‘Validated’ as it represents a validated state with failed Validations used to provide messaging for reporting purposes.

Note: For Price Actions the validation state ‘Calculation Required’ does not require configuration.
