Access To Financial Planning

Go to Plan-->Financial Planning

 

Add Plan

Go to Plan-->Financial Planning-->Manage-->Click on New

 

Initialize Master Plan (Owner)

Go to Plan-->Financial Planning-->Manage-->Select Master Plan and click Initialize

 

Initialize Master Plan (All)

Go to Plan-->Financial Planning-->Manage-->Select Master Plan owned by another user and click Initialize

 

Initialize Branch Plan (Owner)

Go to Plan-->Financial Planning-->Manage-->Highlight branch plan and click Initialize

 

Initialize Branch Plan (All)

Go to Plan-->Financial Planning-->Manage-->Highlight branch plan owned by another user and click Initialize

 

Add Plan Branch

Go to Plan-->Financial Planning-->Manage-->Highlight owned Master plan and click Initialize

 

Merge Plan Branch To Branch (Owner)

Go to Plan-->Financial Planning-->Manage-->Click on initialized owned branch and click to Merge to another Branch

 

Merge Plan Branch To Branch (All)

Go to Plan-->Financial Planning-->Manage-->Click on initialized branch owned by another user and click to Merge to another Branch

 

Merge Plan Branch To Master

Go to Plan-->Financial Planning-->Manage-->Click on owned Brach and click on Merge to Master Plan

 

Manage Global Groups

Go to Plan-->Financial Planning-->Manage-->Click on Global Groups

 

Manage Plan User Access

Go to Plan-->Financial Planning-->Manage-->Click on Plan and click Edit and then click on User Access

 

Edit Master Plan (Owner)

Go to Plan-->Financial Planning-->Manage-->Click on owned plan and then click on Edit

 

Edit Master Plan (All)

Go to Plan-->Financial Planning-->Manage-->Click on plan owned by another user and then click on Edit

 

Edit Branch Plan (Owner)

Go to Plan-->Financial Planning-->Manage-->Click on owned Branch and then click on Edit

 

Edit Branch Plan (All)

Go to Plan-->Financial Planning-->Manage-->Click on Branch owned by another user and then click on Edit

 

Add Plan Snapshot

Go to Plan-->Financial Planning-->Manage-->Click on any plan  and then click on snapshot

 

Delete Plan Snapshot (Owner)

Go to Plan-->Financial Planning-->Manage-->Click on owned plan  with snapshot and then click on Remove

 

Delete Plan Snapshot (All)

Go to Plan-->Financial Planning-->Manage-->Click on plan with snapshot owned by another user and then click on Remove

 

Edit Master Plan Status (Owner)

Go to Plan-->Financial Planning-->Manage-->Click on owned Plan and then click on Status

 

Edit Master Plan Status (All)

Go to Plan-->Financial Planning-->Manage-->Click on Plan owned by another user and then click on Status

 

Edit Branch Plan Status (Owner)

Go to Plan-->Financial Planning-->Manage-->Click on owned Branch and then click on Status

 

Edit Branch Plan Status (All)

Go to Plan-->Financial Planning-->Manage-->Click on Branch owned by another user and then click on Status

 

Add Plan Level

Go to Plan-->Financial Planning-->Manage-->Click on any Plan and click on Edit then click on Product or Channel tabs to Add Level

 

Close Branch Plan (Owner)

Go to Plan-->Financial Planning-->Manage-->Click on Owned Branch and then click on Close

 

Close Branch Plan (All)

Go to Plan-->Financial Planning-->Manage-->Click on Branch owned by another user and then click on Close

 

Close Master Plan (Owner)

Go to Plan-->Financial Planning-->Manage-->Click on owned Master Plan and then click on Close

 

Close Master Plan (All)

Go to Plan-->Financial Planning-->Manage-->Click on Master Plan owned by another user and then click on Close

 

Re-Initialize Plan (Owner)

Go to Plan-->Financial Planning-->Manage-->Click on owned Master Plan and then click on  Re-Initialize

 

Re-Initialize Plan (All)

Go to Plan-->Financial Planning-->Manage-->Click on Master Plan owned by another user and then click on Re-Initialize

 

Re-Seed Branch Plan (Owner)

Go to Plan-->Financial Planning-->Manage-->Click on owned Branch Plan and then click on Re-Seed

 

Re-Seed Branch Plan (All)

Go to Plan-->Financial Planning-->Manage-->Click on Branch owned by another user and then click on Re-Seed

 

Branch Plan Approval

Users can be assigned the rights to approve/reject branch plans has been introduced as part of the Merchandise Financial Planning Module
