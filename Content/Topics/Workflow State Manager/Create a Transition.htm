Create a Transition

Within the Transition screen, click New enter the required fields and click Save.

Field

Description

Transition Name (Optional)

Enter a descriptive label to summarize the purpose of the Status Transition. If left blank, the system will populate the field with ‘Start Node Name -> End Node Name’.

Start Node (Required)

The dropdown is populated with all configured Statuses. Select the Status to start from for a specific Transition.

End Node (Required)

The dropdown is populated with all configured Statuses. Select the ‘next’ Status to end, from the start Status, for a specific Transition.

Users/Roles (Required)

The window is populated with Users and Roles configured with access to the applicable solution. Select the Users and Roles that will have permission to execute the transition from the Start Status to the End Status.

Weight (Optional)

The Weight field can be used, in conjunction with a Status’ Trigger Method where more than one End Node is defined from a single Start Node. Within the Start Node’s Trigger Method, validation criteria should be defined, by JustEnough Professional Services, to determine how the system should transition based on the Weight value.

Note: Weights are required for Order Header Workflows.
