Create a Workflow

Within the Workflow State Management Configuration screen, click New to create a new solution lifecycle, enter and select the required fields and click Save.

Field

Description

Workflow Name (Required)

A descriptive label to summarize the purpose of the workflow lifecycle.

Workflow Description (Optional)

A lengthy explanation of how the workflow lifecycle should be used and how it affects users across the different Statuses.

Workflow Entity Type (Required)

The JustEnough solution that will be associated to the Workflow Status state configuration.

Workflow Exclude Rights (Optional)

The User/Role solution rights that will be controlled by the individual Workflow Status.

Note: If the Exclude Rights field is left blank the system will populate with the expected default set of permissions to control for the selected solution entity type.

Workflow Inactive (Optional)

If a Workflow is marked as ‘inactive’, the system will not make it available in the appropriate solution during creation as well as modification.
