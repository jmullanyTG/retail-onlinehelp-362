Promotion Status States

Promotion Status States are determined by configured Validations that system evaluates against the Promotion during the time of its calculation. For more information on configuring Promotion Validations and viewing failed Validation information, see "Promotion Validations" in Promotion Management Configuration and Validation Statuses in Promotions Home Screen.

  • Validated – All Validations run against the Promotion were successful.
  • Warning(s) – At least one Promotion Validation failed that was configured with the ‘Warning’ severity level.
  • Error(s) – At least one Promotion Validation failed that was configured with the ‘Error’ severity level.

Note: To ease configuration, Promotions Validation Severity type ‘Information’ is included with ‘Validated’ as it represents a validated state with failed Validations used to provide messaging for reporting purposes.

Note: For Promotions the validation state ‘Calculation Required’ does not require configuration.
