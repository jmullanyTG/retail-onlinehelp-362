Restrict User/Role Permissions

To restrict User Rights, select a Status and corresponding State, if applicable. In the Permissions Blacklist grid, click the cell adjacent to the Permission you wish to restrict Users and Roles and the system will display the User/Role Blacklist window. Include, the Users and Roles to remove User Rights during the selected Status State and click OK.

Once all Status States have been configured, click Save, otherwise click Cancel to discard any modifications.
